AnalysisDelegate

@objc public protocol AnalysisDelegate

Delegate which can be used to communicate back to the analysis screen allowing to display custom messages on screen.

Note

Screen API only.
  • displayError(withMessage:andAction:)

    Will display an error view on the analysis screen with a custom message. The provided action will be called, when the user taps on the error view.

    Declaration

    Swift

    func displayError(withMessage message: String?, andAction action: NoticeAction?)

    Parameters

    message

    The error message to be displayed.
    action

    The action to be performed after the user tapped the error view.
  • tryDisplayNoResultsScreen()

    In case that the GiniVisionDocument analysed is an image it will display a no results screen with some capture suggestions. It won’t show any screen if it is not an image, return false in that case.

    Declaration

    Swift

    func tryDisplayNoResultsScreen() -> Bool

    Return Value

    true if the screen was shown or false if it wasn’t.
